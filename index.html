Loans Approval Prediction

Machine Learning Model

Built a supervised machine learning model that predicts loan approval status based on customer data. The model uses Gaussian Naive Bayes along with Scikit-learn, Pandas, and various other machine learning libraries to support the training process. This project demonstrates the end-to-end process of building a predictive model, from data preprocessing to model training, evaluation, and deployment. Additionally, the project includes a random loan application generator that creates reports with as much data as the user wants, allowing for extensive database utilization in the training process. Moreover, the project is equipped with a model that can predict outcomes for new data with high accuracy. You simply need to provide a training dataset, and voilà!

  • Language: Python
  • Concept Used: labeled data, supervised learning model, classification algorithms (Naive Bayes), feature engineering,data preprocessing, model training, and model evaluation.

Employees Promotion Prediction

Machien Learning Model

This project involves developing a machine learning model to predict employee promotion outcomes. The aim is to leverage historical employee data to create a predictive model that forecasts an employee's likelihood of being promoted based on features such as experience, performance, and commitment. The model uses Decision Tree Classifier algorithms along with Scikit-learn, Pandas, and various other machine learning libraries to support the training process. The model is trained on a synthetic dataset generated to provide a diverse set of employee records. It is capable of making accurate predictions on new employee data, assisting in HR decision-making processes.

  • Language: Python
  • Concept Used: labeled data, supervised learning model, Decision Tree Classifier algorithms, feature engineering,data preprocessing, model training, and model evaluation.

Morse Code Converter

Decode the enemy's messages into Englisht

In this Java project, I developed a Morse Code Converter Utility using a generic linked binary tree with generic TreeNodes. The utility translates Morse code into English by constructing a binary tree where each node represents a Morse code symbol or its corresponding English character. I implemented a TreeNode class to serve as the fundamental building block and a MorseCodeTree class to handle the insertion of symbols and traversal for decoding Morse code. Through this project, I gained practical experience in Java programming and learned how to design and implement binary trees using generic programming techniques. This experience enhanced my understanding of hierarchical data structures and their application in solving real-world problems, such as translating Morse code into readable English.

  • Language: Java
  • Concept Used: Generic Classes, Utility Class (all static methods), Linked Trees, ShapeText BoxBuilding a Tree for conversion purposes, loops, data structures.

Mini GPS

Shortest Path between Towns

In this project, I developed an application to manage a network of towns and the roads connecting them, using Dijkstra’s Shortest Path algorithm to find the shortest distance between any two towns. By implementing this algorithm, I created a system capable of efficiently determining the most optimal route within a network of interconnected towns, which enhances route planning and resource management. Through this project, I gained practical experience in implementing graph algorithms and modeling complex networks using graph data structures. I deepened my understanding of algorithmic problem-solving and the application of data structures in real-world scenarios, particularly in optimizing routes and improving efficiency.

  • Language: Java
  • Concept Used: Implement Graph Interface, Use Graph to maintain a network of Vertices, Implement Shortest Path Algorithm

Calendar

Linked List never dies

In this project, I implemented a calendar application designed to schedule events on specific days. The application leverages concepts such as linked lists to manage and organize events efficiently. Users can add, update, and delete events for particular dates, with the calendar dynamically adjusting to reflect these changes. By using linked lists, the application ensures efficient management of events, allowing for quick insertion, deletion, and traversal of the event list. This design choice facilitates smooth and flexible handling of calendar entries, enhancing the overall functionality and user experience of the application.

  • Language: C
  • Concept Used: Linked lists, data structures, calendar management, dynamic data handling, and memory allocation.
